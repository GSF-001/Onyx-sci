# Onyx

Scientific research infrastructure powered by AI.

Onyx is an open-source platform for discovering literature, exploring knowledge graphs, identifying research gaps, tracking emerging trends, and building collaborative research workflows.

---

## Highlights

- AI Research Copilot
- Semantic Search
- Knowledge Graph
- Research Gap Discovery
- Collections
- Collaboration
- Trend Analytics
- Modern Component System
- Type-safe API
- Shared Database Layer

## Philosophy

Onyx is designed around a simple principle:

> Build research software that is modular, type-safe, fast, and maintainable.

Every package has a single responsibility while remaining composable with the rest of the ecosystem.
