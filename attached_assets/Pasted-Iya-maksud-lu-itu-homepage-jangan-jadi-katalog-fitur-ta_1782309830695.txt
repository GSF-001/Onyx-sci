Coba kasih prompt ke AI/Replit gini:

> Redesign the homepage to be search-centric.

Remove all feature cards from the main content area.

The hero section should contain:

Large headline: "Alat Canggih untuk Penelitian Terobosan"

Short description below it

One prominent search box directly underneath the description

Placeholder text: "Cari makalah, konsep, metode, peneliti, atau ajukan pertanyaan..."


Search behavior:

Before the user searches, no papers, results, or recommendations should be displayed.

The homepage remains clean and minimal.

After the user submits a query, dynamically show search results below the search box.

Results should appear as paper cards with title, authors, year, abstract snippet, citation count, and action buttons.

If no query exists, keep the results section hidden.


Layout:

Mobile-first responsive design.

Clean whitespace.

Similar UX philosophy to ChatGPT and Perplexity.

Focus on search, not feature promotion.


Feature navigation:

Keep Semantic Search, AI Copilot, Knowledge Graph, Gap Discovery, Novelty Trends, and Collaboration accessible only through the hamburger menu or sidebar.

Do not display large feature cards on the homepage.


The homepage should feel like a research operating system, not a marketing landing page.
